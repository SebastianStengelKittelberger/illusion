Das Problem

Wir bauen denselben Produktkatalog
immer wieder neu.

🔁

Kunde A, B, C, D …

Jedes Mal: Anforderungen aufnehmen, Datenstruktur analysieren, von Null entwickeln.

🐌

Änderungen dauern Wochen

„Können wir ein neues Feld auf der Produktseite anzeigen?" — Ticket, Sprint, Deployment.

💸

Wachstum durch Köpfe

Mehr Kunden = mehr Entwickler. Keine Plattform bedeutet kein Skalierungseffekt.

Was Kunden wirklich wollen

Kein CMS. Keine Datenbank.
Sie wollen Kontrolle.

Was Kunden sagen:

"Warum steht auf unserer Website immer noch die alte Spannung? Wir haben euch das doch gemeldet."
"Können wir das Bild selbst tauschen ohne euch anzurufen?"
"Wir wissen nicht, woher die Werte auf der Seite eigentlich kommen."

Was Kunden brauchen:

  • Sehen, welcher Wert auf der Seite woher kommt
  • Selbst Felder anpassen — ohne Entwickler
  • Neue Produkte erscheinen automatisch
  • Qualitätsprobleme erkennen bevor Kunden sie melden
Die Idee

Eine Plattform.
Konfigurierbar für jeden Kunden.

illusion verbindet beliebige Produktdaten mit Web-Templates — konfigurierbar, ohne Code. Was früher 6 Monate Entwicklung war, wird in Tagen eingerichtet.

Das Highlight

Der Visual Editor —
kein anderes System kann das.

Der Redakteur sieht die fertige Produktseite.
Er klickt auf einen Wert — z.B. „220V".
Ein Menü öffnet sich: Welches Datenbankfeld soll hier erscheinen?
Er wählt. Die Seite aktualisiert sich sofort.

Kein HTML. Kein Deployment. Kein Entwickler-Ticket.
Mapping-Änderung dauert 30 Sekunden.

Stand heute

Was bereits funktioniert.

  • Mapping-Engine vollständig implementiert
  • Visual Editor mit Klick-zum-Ändern
  • Datenqualitäts-Dashboard
  • Multi-Sprache / Multi-Land
  • Template-System mit Versionshistorie
  • URL-Routing pflegbar ohne Code
  • CMS-Seiten (ohne Produktdaten)
  • Getestet mit realem Bosch-Katalog

Noch offen:

  • Login / Zugriffsschutz
  • Freigabe-Workflow (Entwurf → Live)
  • Self-Service-Onboarding für neue Kunden
  • Produktiv-Deployment

Das System ist ein funktionsfähiger POC.
Kein Spielzeug — eine lauffähige Plattform.

Der entscheidende nächste Schritt

Generic Adapter Wizard —
powered by KI.

Heute braucht jeder neue Kunde einen manuell entwickelten Adapter. Der Generic Adapter Wizard macht das überflüssig: Neue Kunden binden sich selbst an — in Stunden, ohne Entwickler.

📤

Schritt 1: Daten hochladen

REST-API-URL, CSV oder XML — Kunde gibt seine Datenquelle an.

🤖

Schritt 2: KI analysiert

KI erkennt automatisch: "product_voltage" → Spannung. "image_main" → Produktbild.

✅

Schritt 3: Bestätigen & live

Redakteur prüft Vorschläge, korrigiert bei Bedarf — fertig. Produktkatalog ist indexiert.
